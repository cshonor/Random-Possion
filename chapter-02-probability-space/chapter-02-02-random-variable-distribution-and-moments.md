# 第2章-02：随机变量、分布与数字特征

## 随机变量（n维）

`X = (X1,...,Xn): (Ω, F) -> (R^n, B(R^n))` 可测，称为 n 维随机变量（随机向量）。

可测的含义：  
对任意 `B ∈ B(R^n)`，都有 `{ω: X(ω) ∈ B} ∈ F`。

## 分布测度

定义：

`μX(B) = P(X ∈ B) = P(X^{-1}(B)),  B ∈ B(R^n)`

这就是把概率从 `(Ω,F,P)` 搬到 `(R^n, B(R^n))`。

## 分布函数（联合分布）

`F(x1,...,xn) = P(X1<=x1,...,Xn<=xn)`

## 数字特征（2.2）

- 期望 `E[X]`
- 方差 `Var(X)`
- 协方差 `Cov(X,Y)`
- 相关系数

## 随机向量与联合分布（2.3）

- 联合分布 -> 边缘分布
- 联合分布 -> 条件分布
- 多维情形下看“同时发生”的概率

## 本节最短记忆

随机变量的核心不是“有函数式”，而是“可测”。  
可测保证 `P(X∈B)` 始终有定义。
