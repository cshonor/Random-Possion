### 随机变量的独立性（通俗版）

设 `X = (X1, X2, ..., Xn)` 是定义在 `(Ω,F,P)` 上的 n 维随机变量。  
如果对任意 Borel 集 `B1, B2, ..., Bn ∈ B(R)` 都有

```
P(X1∈B1, X2∈B2, ..., Xn∈Bn) = Π_{k=1}^n P(Xk∈Bk)
```

就称 `X1, X2, ..., Xn` 相互独立。

一句话理解：  
**联合概率 = 边缘概率的乘积（并且要对所有可测集合都成立）。**

二维最常用写法：

```
P(X∈A, Y∈B) = P(X∈A)P(Y∈B),   对任意 A,B ∈ B(R)
```

两个小例子：

1. **独立的例子**：两次独立掷骰，`X`=第一次点数，`Y`=第二次点数。  
   因为两次试验互不影响，所以 `X` 与 `Y` 独立。

2. **不独立的例子**：令 `Y = X^2`。  
   `Y` 完全由 `X` 决定，知道 `X` 就知道 `Y`，因此一般不独立。
