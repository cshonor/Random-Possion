# 第2章-06：常用的一维分布速记表

## 1) 二项分布 `B(n,p)`

- **取值**：`k = 0,1,...,n`
- **概率质量函数**：
  `P(X=k) = C(n,k) p^k (1-p)^(n-k)`
- **期望**：`E[X] = np`
- **方差**：`Var(X) = np(1-p)`
- **典型场景**：`n` 次独立重复试验中“成功次数”

## 2) 泊松分布 `P(λ)`

- **取值**：`k = 0,1,2,...`
- **概率质量函数**：
  `P(X=k) = e^(-λ) λ^k / k!`
- **期望**：`E[X] = λ`
- **方差**：`Var(X) = λ`
- **典型场景**：单位时间/单位区域内“稀有事件发生次数”

## 3) 均匀分布 `U(a,b)`

- **定义域**：`x ∈ (a,b)`（常见也写 `[a,b]`，端点不影响连续型概率计算）
- **概率密度函数**：
  `f(x) = 1/(b-a)`（其余为 0）
- **期望**：`E[X] = (a+b)/2`
- **方差**：`Var(X) = (b-a)^2/12`
- **典型场景**：在区间内“等可能”取值

## 4) 指数分布 `Exp(λ)`

- **定义域**：`x >= 0`
- **概率密度函数**：
  `f(x) = λ e^(-λx)`
- **期望**：`E[X] = 1/λ`
- **方差**：`Var(X) = 1/λ^2`
- **典型场景**：Poisson 过程相邻到达时间（等待时间）
- **关键词**：无记忆性

## 5) 正态分布 `N(μ,σ^2)`

- **定义域**：`x ∈ R`
- **概率密度函数**：
  `f(x) = 1/(sqrt(2π)σ) * exp(-(x-μ)^2/(2σ^2))`
- **期望**：`E[X] = μ`
- **方差**：`Var(X) = σ^2`
- **典型场景**：大量独立小扰动叠加后的近似分布（中心极限定理背景）

## 记忆对比（最短版）

- **离散型**：二项、泊松（看 `P(X=k)`）
- **连续型**：均匀、指数、正态（看 `f(x)`）
- **Poisson-指数关系**：计数看 Poisson，等待时间看指数
